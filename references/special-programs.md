# 特殊招生通道参考手册

本文件详细说明高考志愿填报中各类"信息壁垒"通道的政策规则、适用条件和操作要点。

## 一、专项计划体系

### 1.1 高校专项计划

- **招生院校**：教育部确定的95所全国重点高校（含985/211）
- **适用对象**：实施区域农村地区考生
- **三项硬性条件（缺一不可）**：
  1. 符合当年统一高考报名条件
  2. 本人及父亲/母亲/法定监护人户籍在实施区域**农村地区**，本人有当地**连续3年以上户籍**
  3. 本人具有户籍所在县高中**连续3年学籍并实际就读**
- **2026年重大变化**：取消高校自主校测，纯看高考成绩+志愿
- **录取批次**：并入本科提前批
- **兼报限制**：不可与提前批其他类别（军校/公安/航海等）兼报
- **资格审核时间**：6月20日前由各县区完成
- **不可补审**：逾期未通过审核无补救渠道

### 1.2 地方专项计划

- **招生院校**：各省指定的省属重点高校（通常12-15所）
- **条件**：与高校专项基本一致，但竞争相对温和
- **分数线优势**：通常比普通批低10-30分
- **同样并入提前批填报**

### 1.3 国家专项计划

- **招生院校**：中央部门高校和各省属重点高校
- **适用区域**：原集中连片特殊困难县、国家级扶贫开发重点县等
- **注意**：与高校/地方专项的实施区域不同，需核实是否覆盖考生所在县

## 二、提前批特殊通道

### 2.1 军队院校

- 录取在提前批，分数要求因院校和军种而异
- 需通过政审、军检、面试
- 历史类名额少于物理类
- 毕业包分配（军官身份）

### 2.2 公安政法院校

- 需体能测试+面试+政审
- 中国人民公安大学、中国刑事警察学院等
- 毕业参加公安联考，通过率高，入警率高

### 2.3 定向培养医学生

- **农村订单定向医学生**
- 免学费+免住宿费+生活补助
- **毕业后有事业编**（基层卫生院）
- 需签订定向就业协议
- 服务期通常6年
- 适用：宿迁等苏北地区可能有名额

### 2.4 海关相关

- 上海海关学院：海关管理专业毕业可参加海关系统公务员考试（定向招录）
- 分数线较高（历史类通常600+），不适合中分段考生
- 注意：海关管理专业≠进海关保证，但通过率较高

### 2.5 公费师范生

- **2026年江苏已取消公费师范生项目**
- 其他省份需核实当地政策

## 三、中外合作办学/联合培养

### 3.1 核心优势

- 毕业证书与普通专业**完全一致**
- 分数线通常比普通专业低20-50分
- 可获得国内外双学位（部分项目）

### 3.2 学费范围

| 类型 | 学费（每年） |
|------|-------------|
| 中外合作（4+0） | 2.5万~6万 |
| 中外合作（2+2/3+1） | 国内2.5万~5万 + 国外费用 |
| 校企联合培养 | 1万~3万 |
| 中澳/中美学分互认 | 2万~4万 |

### 3.3 适合场景

- 分数低于目标院校普通专业，但家庭经济条件允许
- 希望获得海外学习经历
- 目标院校有明确的中外合作项目

### 3.4 注意事项

- 部分项目要求外语单科成绩
- 录取后通常不可转入普通专业
- 需确认项目是否经教育部审批（避免"野鸡"项目）

## 四、征求志愿/征集志愿捡漏

### 4.1 机制

- 各批次录取结束后公布未完成招生计划的院校
- 考生在规定时间内上网填报（通常仅1天窗口）
- 线上生源不足可降分投档，**最大降分不超过20分**

### 4.2 常见断档原因

- 新增院校/专业首年招生，考生不了解
- 地理位置偏远（西北、西南）
- 冷门专业（小语种、哲学、某些农林专业）
- 中外合作项目学费高导致报考人数不足
- 征求志愿窗口短，部分考生错过

### 4.3 历史真实捡漏案例

- 宁夏考生433分录取上海交通大学（专项计划征集志愿降分）
- 西藏大学在多省断档，降30分录取
- 某985高校波斯语专业无人报考，跌至一本线附近

### 4.4 操作要点

- 密切关注省教育考试院官网的缺额公告
- 提前预判可能断档的院校/专业
- 征求志愿填报时间极短（可能仅几小时），需提前准备志愿清单

## 五、转专业通道

### 5.1 适用场景

- 分数不够目标院校热门专业
- 但能录取到该校的冷门/低分专业
- 入学后通过转专业进入意向专业

### 5.2 常见转专业条件

| 条件类型 | 说明 |
|---------|------|
| GPA要求 | 通常要求大一成绩排名前10%-30% |
| 考试要求 | 部分学校需通过转专业考试 |
| 名额限制 | 每个转入专业有名额上限 |
| 时间窗口 | 通常在大一/大二第一学期申请 |

### 5.3 操作要点

- **志愿填报前必须电话确认**目标院校的转专业政策
- 确认意向专业是否接受转入、转入名额多少
- 部分院校对转专业有先修课程要求
- 优先选择转专业政策宽松的院校

## 六、加分政策

### 6.1 全国统一加分项目

| 项目 | 加分 |
|------|------|
| 烈士子女 | +20分 |
| 荣立二等功以上退役军人 | +20分 |
| 归侨/华侨子女/归侨子女 | +5分 |
| 台湾省籍考生 | +5分 |
| 自主就业退役军人 | +10分 |
| 少数民族（仅限特定区域） | +5分（江苏无此政策） |

### 6.2 注意事项

- 加分仅限于投档环节，高校录取时可能折算或不承认
- 同一考生最多享受一项加分，不累加
- 2026年起加分政策进一步收紧，需核实最新规定
