# 院校专业匹配策略参考手册

## 一、匹配核心原则

### 1.1 位次优先于分数

- 分数每年波动，位次相对稳定
- 必须同时参考分数和位次进行匹配
- 使用**前两年数据交叉验证**

### 1.2 专业级差与专业组

- 新高考省份使用"院校专业组"模式
- 2026年江苏等省份专业组大幅拆分，冷热分离
- 同一院校不同专业组分数线差异可能很大
- 需关注每个专业组的选科要求

### 1.3 平行志愿投档规则

- "分数优先、遵循志愿、一轮投档"
- 前面的志愿优先投档
- 一旦投档成功，后续志愿作废
- **滑档风险**：如果所有志愿都填得过高且不服从调剂，可能直接掉档

## 二、搜索数据的标准流程

### 2.1 分数线数据搜索

使用 `search` 工具搜索时，关键词模板：

```
[院校名] 2025 [省份] [科类] [专业名] 录取分数线
[院校名] 2025 [省份] [科类] 最低录取分 位次
[院校名] 2025 [科类] 录取专业分 [省份]
```

示例：
```
南京审计大学 2025 江苏 历史类 审计学 录取分数线
山西财经大学 2025 江苏 历史类 会计学 录取分数线
```

### 2.2 数据源优先级

| 优先级 | 数据源 | 特点 |
|--------|--------|------|
| 1 | 省教育考试院 | 最权威，官方数据 |
| 2 | 目标院校招生网 | 专业级分数线最详细 |
| 3 | 志愿填报网站（阳光高考、掌上高考等） | 汇总数据，方便对比 |
| 4 | 百度搜索结果 | 快速预览，需二次验证 |

### 2.3 使用 browser 获取精确数据

当 search 结果不够精确时，使用 browser 工具访问：
- 目标院校招生网 → 历年录取数据
- 省教育考试院 → 各批次投档线
- 掌上高考/志愿填报APP网站 → 专业分数线查询

## 三、按专业大类的匹配策略

### 3.1 财经类（会计/金融/审计）

**院校梯队**：
| 梯队 | 院校类型 | 代表院校 |
|------|---------|---------|
| 顶尖 | 财经类顶尖 | 上财、央财、对外经贸、中南财 |
| 强势 | 老牌财经 | 南审、南财、西财、东财、江财、山财 |
| 省属 | 省属财经 | 天津商大、广东金融学院、河南财经政法 |
| 普通 | 综合大学财经 | 各省综合大学经济管理学院 |

**外省信息差策略**：
- 江苏考生 → 山西财经、天津商大、广东金融学院
- 山东考生 → 江西财经、南京审计（外省视角）
- 核心逻辑：考生不愿出省 → 外省同层次院校分数线被压低

**中外合作项目**：
- 南京审计大学中外合作：比普通专业低40-50分
- 适合分数在特招线附近、想进名校的考生

### 3.2 师范类

**路径**：
- 公费师范生（部分省份2026年已取消，需核实）
- 普通师范类院校
- 综合大学的师范专业
- 入学后考教资

### 3.3 医学类

**路径**：
- 定向培养医学生（有事业编，需签约服务）
- 普通医学院校
- 综合大学医学专业（通常分数较高）

### 3.4 工科类

**路径**：
- 传统工科强校
- 新兴工科（人工智能、大数据等）
- 注意选科要求（通常需要物理+化学）

## 四、冲稳保梯度配置

### 4.1 标准配置比例

| 梯度 | 比例 | 说明 |
|------|------|------|
| 冲 | 20%-30% | 录取概率15%-30% |
| 稳 | 40%-50% | 录取概率50%-70% |
| 保 | 20%-30% | 录取概率80%+ |

### 4.2 调整因素

- **风险偏好型**：增加"冲"的比例，减少"保"
- **保守型**：增加"保"的比例，减少"冲"
- **有信息差机会**：可将信息差院校作为"冲"选项

### 4.3 服从调剂

- 建议勾选"服从调剂"以降低滑档风险
- 但需注意：被调剂到不满意专业的风险
- 如不服从调剂，需确保保底志愿足够稳

## 五、志愿填报注意事项

### 5.1 时间节点

| 时间 | 事项 |
|------|------|
| 6月24-25日 | 各省公布分数线 |
| 6月20日前 | 专项计划资格审核截止 |
| 6月28日-7月2日 | 第一阶段志愿填报（江苏） |
| 7月上旬 | 提前批录取 |
| 7月中下旬 | 本科批录取 |
| 7月下旬 | 征求志愿填报（缺额补录） |
| 8月 | 专科批录取 |

> 具体时间因省份而异，需以省教育考试院通知为准。

### 5.2 常见失误

| 失误 | 后果 | 预防 |
|------|------|------|
| 所有志愿都冲高 | 滑档 | 必须有保底志愿 |
| 不服从调剂 | 退档 | 除非确保分数足够 |
| 忽视选科要求 | 无法投档 | 逐一确认专业组选科要求 |
| 错过征求志愿 | 失去捡漏机会 | 密切关注考试院公告 |
| 提前批/普通批兼报冲突 | 志愿作废 | 确认兼报规则 |

### 5.3 特殊情况处理

- **征集志愿降分**：最多降20分，需在窗口期内填报
- **预科班**：少数民族、边防子女等可走预科通道（降分录取，多读1年预科）
- **复读了**：复读生与应届生同等待遇（部分军校/公安院校限制复读生报考）
