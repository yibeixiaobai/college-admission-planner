# 高考志愿填报智能规划技能

为考生提供数据驱动的志愿填报方案，核心优势是**挖掘信息差机会**，帮助考生用同等分数读更好的学校/专业。

## 输入参数

向用户确认以下信息（未提及则逐项询问）：

| 参数 | 说明 |
|------|------|
| 省份 | 考生所在省份（必填） |
| 科类/选科 | 文/理/历史类/物理类/新高考选科组合（必填） |
| 高考分数 | 2026年高考总分（必填） |
| 专业意向 | 目标专业或专业大类（必填） |
| 城市偏好 | 留省内/出省/指定城市（可选） |
| 特殊条件 | 农村户籍/少数民族/军人子女等（可选） |
| 家庭预算 | 中外合作项目学费承受力（可选） |
| 上升需求 | 是否需要"上一个台阶"/信息壁垒机会（可选） |

## 工作流程

### 第一步：分数定位

1. 使用 `search` 工具搜索该省份**当年（2026年）各批次分数线**，包括：
   - 本科线/一本线/二本线
   - 特殊类型招生控制线（特招线）
   - 一分一段表或位次对照表
2. 计算考生分数与各批次线的分差
3. 根据分数和位次，确定考生所在**分数段层级**：
   - 顶尖段（985层级）
   - 中上段（211/双一流层级）
   - 中段（省属重点/区域强校）
   - 中下段（公办本科普通校）
   - 保底段（民办/职业本科）

输出格式：
```
分数：XXX分
批次线：XXX分（超线XX分）
特招线：XXX分（超线XX分）
预估位次：约X万~X万名
层级定位：[层级名称]
```

### 第二步：信息差机会挖掘（核心差异化能力）

**按优先级检查以下上升通道，向用户说明哪些适用、哪些不适用及原因：**

#### 通道1：专项计划（高校专项/地方专项/国家专项）
- 使用 `search` 搜索该省份当年专项计划政策
- 检查条件：农村户籍+连续3年户籍+当地高中学籍
- 检查该省是否在实施区域
- 2026年起高校专项不再校测，纯看高考成绩+志愿
- 注意：高校专项/地方专项并入提前批，不可与军校/公安等兼报
- 参考 [references/special-programs.md](references/special-programs.md) 了解各类专项计划详情

#### 通道2：中外合作办学/联合培养
- 搜索目标院校中外合作项目历年分数线
- 毕业证与普通专业完全一致，用学费换分数（每年2.5万~6万）
- 适合分数略低于目标院校普通专业的考生
- 关键数据：搜索"XX大学 中外合作办学 2025 [省份] [科类] 录取分数线"

#### 通道3：外省院校信息差
- 核心逻辑：本省考生不愿出省 → 外省优质院校分数线被压低
- 特别适用于财经/师范/医学类院校
- 搜索外省同层次财经/师范/医学院校在该省录取分
- 回本省就业时，用人单位看重专业能力+证书，非学校所在地

#### 通道4：征求志愿（征集志愿）捡漏
- 各批次录取结束后组织，缺额院校降分录取（最多降20分）
- 关注时间：本科批录取期间（约7月中下旬）
- 信息来源：省教育考试院官网公告
- 策略：提前锁定可能断档的院校/专业，第一时间填报

#### 通道5：转专业通道
- 填报目标院校的冷门/低分专业组进门
- 入学后转至意向专业
- 必须提前电话确认目标院校转专业政策
- 2026年专业组拆分后，冷热分离更容易操作

#### 通道6：提前批特殊通道
- 军校、公安、海关、航海、免费师范（2026江苏已取消公费师范）
- 定向培养医学生（有事业编）
- 边防子女预科班、少数民族预科班
- 注意时间线：提前批志愿在普通批之前

### 第三步：院校专业匹配

1. 使用 `search` 搜索目标专业在各院校的**录取分数线和位次**：
   - 搜索关键词："XX大学 [专业名] 2025 [省份] [科类] 录取分数线"
   - 搜索关键词："XX大学 2025 [省份] [科类] 最低录取分 位次"
2. 使用 `browser` 工具访问具体院校招生网或志愿填报数据网站获取精确数据
3. 收集至少15-20所院校的分数线数据
4. 参考 [references/matching-strategy.md](references/matching-strategy.md) 了解匹配策略

### 第四步：生成冲稳保方案

将匹配到的院校分为三个梯度：

**冲一冲（录取概率15%-30%）**：
- 分数线高于考生分数0-15分的院校/专业
- 利用信息差通道的冲刺院校
- 中外合作项目中的冲刺目标

**稳一稳（录取概率50%-70%）**：
- 分数线与考生分数持平或低0-10分的院校/专业
- 外省信息差院校
- 省内优质公办本科的匹配专业

**保一保（录取概率80%+）**：
- 分数线低于考生分数10-20分的院校/专业
- 本地公办本科
- 职业本科（2026年扩招主力）

**输出格式**：使用 Markdown 表格，每所院校包含：

| 梯度 | 院校 | 专业 | 2025参考分 | 分析/策略 |
|------|------|------|-----------|-----------|
| 冲 | XX大学 | XX专业 | XXX分 | 说明 |
| 稳 | XX大学 | XX专业 | XXX分 | 说明 |
| 保 | XX大学 | XX专业 | XXX分 | 说明 |

### 第五步：综合方案输出

生成完整的志愿规划报告，包含：

1. **分数定位表**（第一步输出）
2. **信息差机会分析**（第二步输出，说明哪些可用、哪些不可用）
3. **冲稳保志愿方案表**（第四步输出）
4. **关键时间节点提醒**
5. **行动清单**（按优先级排序的待办事项）
6. **注意事项与风险提示**

报告格式要求：
- 使用 Markdown 格式
- 表格清晰、层次分明
- 策略说明简明扼要
- 避免冗余解释

## 搜索失败处理

当 `search` 无法获取数据时，按以下顺序降级处理：

1. **换关键词重试**：调整搜索词（如"XX大学 2025 录取分"→"XX大学 2024年分数线"）
2. **换数据源**：从省教育考试院 → 院校招生网 → 志愿填报网站逐级尝试
3. **使用 browser 访问**：直接打开目标网站获取数据
4. **告知用户**：明确说明哪些数据缺失，提供参考范围而非编造数据

> 若多项关键数据均无法获取，应在报告中标注"数据待核实"，建议用户通过省教育考试院官网或电话咨询确认。

## 能力边界

本工具**可以**：
- 基于历史数据提供志愿填报参考方案
- 挖掘信息差机会（中外合作、外省院校、专项计划等）
- 生成冲稳保梯度方案

本工具**不能**：
- 保证录取结果（当年分数线可能波动）
- 替代考生和家长的最终决策
- 提供实时数据（需以当年公布数据为准）
- 替代人工核实关键政策信息（各省政策差异大）

## 常见问题（FAQ）

**Q: 为什么搜索不到某院校的分数线？**
A: 部分院校数据公开程度有限，可尝试搜索"XX大学 招生网"访问其官方数据，或使用掌上高考等第三方平台查询。

**Q: 信息差机会真的靠谱吗？**
A: 信息差机会基于真实政策和历年数据，但每年情况可能变化。中外合作项目确实存在分数优势，但需确认项目是否经教育部审批。

**Q: 中外合作项目值得报吗？**
A: 适合家庭经济条件允许、分数略低于目标院校普通专业的考生。毕业证与普通专业一致，但需注意部分项目不可转入普通专业。

**Q: 冲稳保比例怎么调整？**
A: 风险偏好型可增加"冲"的比例（30%+），保守型可增加"保"的比例（40%+），详见 [references/matching-strategy.md](references/matching-strategy.md)。

**Q: 各省政策差异大，如何确保准确？**
A: 本工具提供通用框架，具体政策（如加分、专项计划区域、提前批规则）需以考生所在省教育考试院官方发布为准。

## 关键数据来源

| 数据类型 | 来源 | 工具 |
|---------|------|------|
| 分数线/位次 | 省教育考试院、阳光高考网 | `search` |
| 院校录取分 | 各校招生网、志愿填报网站 | `search` + `browser` |
| 专项计划政策 | 教育部、省教育考试院 | `search` |
| 中外合作项目 | 各校招生网、教育部中外合作办学监管平台 | `search` + `browser` |
| 征求志愿缺额 | 省教育考试院官网 | `browser`（实时监控） |
| 转专业政策 | 各校教务处/招生办 | `search` 或电话确认 |

## 注意事项

- 所有分数线数据必须标注年份（默认参考2025年，2026年数据出来后以最新为准）
- 位次比分数更稳定，应同时参考位次
- 新高考省份注意选科要求（物理/历史+化学等）
- 中外合作项目须注明学费范围
- 提前批与普通批的兼报规则因省而异，需核实
- 志愿填报方案仅供参考，最终决定权在考生和家长

## 示例：完整志愿规划流程

**输入**：江苏考生，历史类，580分，想学财经类专业，愿意出省

**输出摘要**：

```
分数：580分
批次线：474分（超线106分）
特招线：527分（超线53分）
预估位次：约1.2万名
层级定位：中上段（211/双一流层级）
```

**信息差机会**：
- 专项计划：不适用（非农村户籍）
- 中外合作：南京审计大学中外合作可冲（比普通专业低40分）
- 外省院校：山西财经大学、天津商业大学可作为"稳"选项

**冲稳保方案**：

| 梯度 | 院校 | 专业 | 2025参考分 | 策略 |
|------|------|------|-----------|------|
| 冲 | 南京审计大学 | 审计学（中外合作） | 590 | 用学费换分数，毕业证一致 |
| 冲 | 江西财经大学 | 会计学 | 588 | 外省信息差，财经强校 |
| 稳 | 山西财经大学 | 金融学 | 575 | 外省降分录取，性价比高 |
| 稳 | 天津商业大学 | 财务管理 | 570 | 老牌财经，外省考生少 |
| 保 | 广东金融学院 | 会计学 | 560 | 省属财经，就业认可度高 |
| 保 | 河南财经政法大学 | 审计学 | 555 | 保底公办本科 |
